## About LeetCode

O [LeetCode](https://leetcode.com) é uma das principais plataformas para praticar problemas de programação e se preparar para entrevistas técnicas. Ele oferece uma vasta coleção de desafios que ajudam desenvolvedores a aprimorar suas habilidades em algoritmos.

## Visão geral do projeto
Este repositório é mantido por [@Vitor4818](https://github.com/Vitor4818) como parte de uma rotina de prática de longo prazo focada em melhorar a resolução de problemas, estruturas de dados e pensamento algorítmico.

O objetivo deste projeto é resolver pelo menos um problema do LeetCode por dia, organizando as soluções por dificuldade e tópico, mantendo código limpo, disciplina de commits e aprendizado contínuo. Cada solução inclui uma implementação clara e bem estruturada e, quando relevante, uma explicação da abordagem utilizada.

## Estrutura de diretórios

Cada novo problema do LeetCode é organizado em uma pasta dedicada dentro do diretório `problems`. A estrutura segue um padrão simples para manter as soluções limpas e fáceis de navegar:
```
problems/
└── Easy/
│ └── TwoSum/
│ ├── Solution.java
│ ├── Main.java
│ └── README.MD
```

## Estrutura do repositório
Cada problema está organizado em seu próprio diretório, contendo:
- **description.md**: Enunciado do problema.
- **explanation.md**: Uma explicação detalhada da abordagem e da lógica utilizada.
- **solution.Java**: A solução implementada em Java.
